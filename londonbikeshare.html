London Bike Sharing Program

Overview

Santander Cycles sponsors London's government run bike share program. Transport for London provides open source datasets encouraging software developers to use these feeds to present customer travel information in innovative ways.

This interactive Tableau dashboard quickly visualizes the usage of bike share program at a given hour of the day.
Hover over the heatmap to view details.
